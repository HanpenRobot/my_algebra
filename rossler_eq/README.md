# Rossler方程式を4次のルンゲ・クッタ法で解いた結果の3Dプロット
+ 下記のRossler方程式を$`(a,b,c)=(0.2,0.2,5.7)`$、初期値$`x(0)=0.3, y(0)=0, z(0)=0`$の条件で4次のルンゲ・クッタ法で解き、その結果を3Dプロットした。

+ 微分方程式を解く際に使用したルンゲ・クッタ法のコードは[./runge_kutta_3d.c](./runge_kutta_3d.c)である。 (このコードは参考文献[2]のコードを参考に実装した)。
```math
\frac{d x}{d t} = -(y+z) \cdots (1)
```
```math
\frac{d y}{d t} = x+ay \cdots (2)
```
```math
\frac{d z}{d t} = b+z(x-c) \cdots (3)
```
