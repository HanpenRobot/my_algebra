# 超離散化した拡散方程式
+ 超離散化した拡散方程式を以下に示す。
```math
U_{i,j,n}=\max({U_{i,j,n},U_{({i+1}),j,n}U_{({i-1}),j,n},U_{i,({j+1}),n},U_{i,({j-1}),n},U_{i,j,({n-1})}})-U_{i,j,({n-1})} \cdots (1)
```
