# 区分線形連立微分方程式を4次のルンゲ・クッタ法で解いた結果の3Dプロット
+ 下記の区分線形連立微分方程式を$`(a1,a2,a3)=(0.19,-1.9644,6.3122), (c1,c2,c3)=(-2.69,-2.383,-14.049)`$、初期値$`x(0)=0.3, y(0)=0, z(0)=0`$の条件で4次のルンゲ・クッタ法で解き、その結果を3Dプロットした(区分線形連立微分方程式のパラメータ$`(a1,a2,a3), (c1,c2,c3)`$は参考文献[1]の値を利用した)。

+ 微分方程式を解く際に使用したルンゲ・クッタ法のコードは[./runge_kutta_3d_piecewise_linear.c](./runge_kutta_3d_piecewise_linear.c)である。 (このコードは参考文献[2]のコードを参考に実装した)。
```math
\frac{d x}{d t} = c_1 x + y + \frac{1}{2} c_1 (|x-1|-|x+1|) \cdots (1)
```
```math
\frac{d y}{d t} = c_2 x + z + \frac{1}{2} c_2 (|x-1|-|x+1|) \cdots (2)
```
```math
\frac{d z}{d t} = (c_3+a_3) x + a_2 y + a_1 z + \frac{1}{2} c_3 (|x-1|-|x+1|) \cdots (3)
```
