# エノン写像を繰り返し写像した結果のプロット
+ 下記の区分線形連立微分方程式を$`(a,b)=(1.4,0.3)`$、初期値$`x(0)=0.01, y(0)=0.01`$の条件でエノン写像を繰り返し写像しその結果をプロットした(エノン写像のパラメータ$`(a,b)=(1.4,0.3)`$は参考文献[1]の値を利用した)。

+ エノン写像を繰り返し写像した値を求めるコードは[./henon_map.c](./henon_map.c)である。
```math
x=y+1-ax^2\cdots (1)
```
```math
y=bx \cdots (2)
```
