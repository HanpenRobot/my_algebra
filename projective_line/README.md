1次元射影直線$`RP^1`$の定義
```math
RP^1=原点を通る直線の集合
```
つまり、$`RP^1`$は
```math
R^2=\{(x_0,x_1) | x_0, x_1 \in R \}
```
に以下の同値関係$`\sim`$
```math
R^2 \setminus \{ 0 \} \ni (x_0,x_1) = \alpha (x_0',x_1') \iff (x_0,x_1) \sim (x_0',x_1')
```
をいれた時の商集合
```math
RP^1 = R^2 / \sim
```
である。
